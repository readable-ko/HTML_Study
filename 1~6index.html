Description list 정의 리스트 수업내용

수업 내용

dl은 2가지 사용법이 있는데 dl안에는 dt dd div만 직계자식으로 사용가능!

  1. 용어의 대한 정의를 할 때
  2. key-value로 정보를 제공할 때 ex: 고영광: 한동대 사망년
  • dl은 정의 리스트를 의미
  • dt는 description term 이름(용어, key 값)에 대항하는 것을 마크업할 때 사용
  • dt를 사전적으로 용어를 정의하고 싶을 때 definition의 의미로 dfn 를 사용
  • dd는 Description data 를 의미하고 key를 설명하는 부분!

수업 심화 내용

OL: ordered list 순서가 중요한 리스트
ex: 요리방법, 실검 UL: unordered list 순서가 중요하지 않은 리스트
ex: 순서가 없는 것들
ol과 ul 모두 안에 있는 항목을 나열할 때는 list item li를 사용한다 여기서 지켜야하는 문법은 ul과 ol안에는 div p a 같은 것 사용이 불가능하며 무조건 li만 사용가능하다
